# Shiba Runner

A retro-inspired static browser game.  
Click or tap, or press `Space` / `W` / `Up`, to jump over obstacles, grab coins, and push for a new hi-score.

## Features

- No build step required
- Static `index.html` project, easy to upload to GitHub Pages
- Local hi-score persistence
- Touch and keyboard controls
- Bundled SVG assets included in the repository

## Run Locally

Open `index.html` in a browser.

You can also preview it with any static file server if you prefer.

## Publish On GitHub Pages

1. Upload this folder to a new GitHub repository.
2. Open `Settings > Pages`.
3. Choose `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Wait a few minutes for the public URL.

## File Layout

- `index.html`: entry point
- `styles.css`: layout and presentation
- `src/game.js`: gameplay logic
- `assets/`: bundled SVG art assets
